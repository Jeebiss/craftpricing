CraftPricing

For Minecraft 1.2.4 - By viveleroi of dhmc.us

A javascript-based pricing engine for all items in Minecraft.

By entering pricing information for all base materials, this program will automatically generate pricing data for all other items.

To keep this application lightweight and easily distributable, the data is stored in raw JSON format, in the /js/recipe data.

The initial package comes with a set of unit tests for both data structure and price calculations. Adjust the tests expected values and run to ensure your data is complete. You can pass in your own callback formatter to describe the method of data output. An example formatter is provided, exporting the data into MySQL INSERT queries.
